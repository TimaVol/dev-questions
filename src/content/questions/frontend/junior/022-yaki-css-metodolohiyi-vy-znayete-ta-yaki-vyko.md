## Відповідь

BEM (блок__елемент--модифікатор), OOCSS, SMACSS, ITCSS. BEM найпоширеніший — дає передбачувані імена класів і зменшує конфлікти специфічності в великих проєктах.

## Приклад

```css
.card { padding: 1rem; }
.card__title { font-size: 1.25rem; }
.card--featured { border: 2px solid gold; }
```

## Юз кейси

- Єдиний стиль іменування в команді з 5+ розробників
- Компонентна бібліотека, де класи не перетинаються між модулями
