## Відповідь

Дивись на: розмір і досвід команди, вимоги до SEO (SSR), екосистему, вакансії на ринку, performance, наявність дизайн-системи. Не вибирай фреймворк «бо модно» — під задачу проєкту.

## Приклад

```jsx
// Next.js — якщо потрібен SSR/SSG з React
// Nuxt — аналог для Vue
// Astro — контентні сайти з мінімальним JS
```

## Юз кейси

- E-commerce з SEO — Next.js/Nuxt з SSR
- Internal admin panel — React SPA достатньо
